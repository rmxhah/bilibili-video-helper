# 视频下载助手 V3.1

> 多平台视频/字幕/音频下载工具，支持画质选择

## 功能

- 🔐 **B站扫码登录** — 手机B站App扫码即用
- 📝 **字幕下载** — SRT + 纯文本，原生CC字幕优先，AI字幕兜底
- 🎵 **音频下载** — 独立音频流
- 🎬 **视频下载** — 全画质可选，B站DASH+ffmpeg合并，YouTube/抖音直链
- ⚡ **多线程加速** — 分块并行下载
- 🎯 **三平台** — B站 / YouTube / 抖音
- 🔍 **代理自动检测** — YouTube自动读取系统代理

## 快速开始

1. 双击 `BiliHelper.exe`
2. B站：点击「扫码登录」→ 手机B站App扫码
3. 粘贴视频链接，选择画质和输出类型
4. 点击「开始处理」

## 系统要求

- Windows 10/11 64位
- 无需安装Python或任何依赖
- YouTube需要代理（支持HTTP代理/TUN模式，自动检测）

## ⚠️ Windows安全中心提示

本程序为开源Python打包，**不含任何恶意代码**。

由于未购买商业代码签名证书，Windows Defender可能误报。解决方法：

1. 打开 **Windows安全中心** → **病毒和威胁防护** → **保护历史记录**
2. 找到被拦截的 `BiliHelper.exe` → 点击 **操作** → **允许**
3. 或者将程序文件夹添加到 **排除项**：
   - **病毒和威胁防护** → 管理设置 → **排除项** → 添加文件夹
   - 添加 `视频下载助手发布版3.0` 整个文件夹

**为什么误报？**
- Python打包工具(PyInstaller)生成的自解压EXE会被启发式扫描误判
- 未签名的EXE默认被SmartScreen标记为"未知发布者"
- 已通过禁用UPX压缩、添加版本元数据等方式降低误报率

## 隐私说明

- `config.json` 是程序首次运行后自动生成的，分享给他人前请删除
- 分享时请删除 `config.json` 文件
